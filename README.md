# Short Description

The content of the particular project consists of archaeological data, that are stored in a data base and they are presented to the user through webpage, which is compatible with all electronic devices.
The **XAMPP** platform was used, the **VisualParadigm** for the data base creation, the data managing system used was **MySQL**. The languages and frameworks used were **PHP**, **HTML**, **CSS**, **Bootstrap** and **jQuery**.

# ER Model 
#### Archaeological

Archaelogical ER shows the precise geographical locations identified by the Nomenclature of Territorial Units for Statistics (NUTS), serving as the backdrop for excavation sites. Comprehensive environmental information surrounding these excavation sites is also depicted. Moreover, the diagram encapsulates details about discovered artifacts (in sites), including their respective inscriptions, seamlessly linked to specified chronological dates.

### Ethnographic 

The Ethnographic ER diagram encompasses tables tailored for the storage of archaeological data gathered from local communities during interviews. Specifically designed to capture the insights provided by interviewees, this ER diagram includes tables aptly suited for organizing and preserving the nuanced details shared by local people regarding archaeological information.
